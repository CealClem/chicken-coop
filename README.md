# 🐔 Chicken Coop Manager

A web app (with mobile planned) to coordinate shared responsibilities for a community chicken coop.

## What it does

Members of a shared chicken coop can claim day and evening care slots on a monthly calendar. Clicking any slot lets you enter a name to assign it, or clear it if plans change.

## Planned

- Mobile app (Android/iOS)
- Persistent storage so assignments survive a page refresh
- User accounts so members can manage their own slots
- Reminder notifications for assigned slots

## Running locally

```bash
npm install
npm run dev
```
